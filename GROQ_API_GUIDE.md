## Quick Setup

### Step 1: Get a Free API Key
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up (free account)
3. Navigate to "API Keys"
4. Create a new API key
5. Copy the key (keep it secret!)

### Step 2: Start TermChat in AI Mode
1. Open the application
2. Click button **[2] API** at boot screen
3. Paste your API key when prompted
4. Press Enter

### Step 3: Use /ai Command
```
/ai What is machine learning?
/ai Write a haiku about programming
/ai Explain quantum computing briefly
```

## Troubleshooting

### "Invalid API key"
- Check you copied the full key correctly
- Verify key hasn't expired (check Groq console)
- Try creating a new key

### "Rate limited"
- Too many requests too quickly
- Wait a few seconds and try again
- Free tier has limits

### "Network error"
- Check internet connection
- Groq API might be down (rare)
- Try Local AI mode instead

### Want to Switch Modes?
- Refresh the page (F5)
- Click a different boot button
- Chat will start fresh

## Local AI Mode (No Key Needed)

Don't have an API key? Use Local AI mode:
1. Click button **[3] LOCAL** at boot screen
2. AI will generate simulated responses
3. No API key required
4. Use same `/ai` commands

## Pro Tips

1. **Keep responses brief** - Ask specific questions
2. **Use technical language** - AI responds better to detailed prompts
3. **Multi-language** - Try asking in any language
4. **Build on answers** - Continue conversation naturally
5. **Save your key** - Copy it to browser localStorage (already done!)
